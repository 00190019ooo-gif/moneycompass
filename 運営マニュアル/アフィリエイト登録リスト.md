## 収益シミュレーション（現実的）

```
月間訪問者 10,000人の場合：

クリック率（CTR）: 3% → 300クリック
申込み率（CVR）: 5% → 15件申込み
報酬単価: 8,000円

月収 = 15件 × 8,000円 = 120,000円
```

```
月間訪問者 50,000人の場合：

クリック率（CTR）: 3% → 1,500クリック
申込み率（CVR）: 5% → 75件申込み
報酬単価: 8,000円

月収 = 75件 × 8,000円 = 600,000円
```

## Boss がやること（一回だけ）

1. **A8.net** にアクセスして「メディア会員として登録」
2. **afb** にアクセスして登録
3. 登録完了後、メールで届くIDとパスワードをSECに共有
4. あとはSEC・FINが各プログラム参加・リンク管理を全部やる

*サイトURLは `moneycompass.jp`（または取得したドメイン）を登録する*
*サイトの審査は通常1〜3日かかる*

## 重要：広告表示の規則

金融アフィリエイトは景表法・金融広告規制の対象。必ず以下を守る：

- 「必ず借りられる」「審査なし」等の断定的表現はNG
- アフィリエイトリンクには「広告」「PR」の表示が必須
- 金利・条件は必ず最新情報に更新する（GROW班が月1回チェック）

*これをCHECK班が毎月監査する*
