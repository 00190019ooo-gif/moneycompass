# Boss が今すぐやること
**優先順に並べてある。上から順番にやるだけ。**

---

## 今日やる（合計30分以内）

### 1️⃣ ドメインを取る（10分・約1,000円/年）

1. **お名前.com** を開く（onamae.com）
2. 検索窓に `moneycompass` と入力
3. `.jp` か `.com` を選んで購入
4. ※取れなければ `money-compass` や `moneynavi` でもOK

---

### 2️⃣ Slack を作る（5分・無料）

1. **slack.com** を開く
2. 「Slackを無料で試す」をクリック
3. ワークスペース名：**マネーコンパス**
4. 以下のチャンネルを作成：
   - `boss-commands`
   - `reports`
   - `alerts`
5. 完成したら俺に「Slack作ったよ」と教えて

---

### 3️⃣ Claude API キーを取得する（5分・無料）

1. **claude.ai** にアクセス（今使ってるClaudeの公式サイト）
2. 右上のアカウント → 「APIキー」または Settings
3. 「Create Key」でキーを作成
4. キーをメモ帳に保存しておく（あとでMake.comに入れる）

---

### 4️⃣ Make.com に登録する（5分・無料）

1. **make.com** を開く
2. 「Get started free」で無料登録
3. ログインしたら俺に「Make登録した」と教えて

---

### 5️⃣ A8.net に登録する（5分・無料）

1. **a8.net** を開く
2. 「メディア会員として登録」
3. サイトURL欄：後で取得するドメインを入力（なければ仮でOK）
4. 登録完了後、俺に「A8登録した」と教えて

---

## 来週やる（合計60分以内）

### 6️⃣ Xserver でサーバーを借りる（15分・月1,100円〜）

1. **xserver.ne.jp** を開く
2. 「お試し無料10日間」で申込み
3. 取得したドメインと紐づける作業は俺がガイドする
